## C1 — PostgreSQL Double-entry Ledger

สถานะ: **IMPLEMENTED / VERIFIED (LOCAL-ONLY)**

### สิ่งที่ทำ

- เพิ่ม `Backend/database` แยกจาก Unity project พร้อม ordered migrations และ test harness
- เพิ่ม currencies: Gold, War Gem และ Premium Diamond; Premium Diamond ไม่ใช่ raid stake
- เพิ่ม ledger accounts/transactions/postings, API idempotency store และ transactional outbox
- เพิ่ม system issuance/sink accounts และ `post_ledger_transaction(...)`
- ใช้ SHA-256 request hash + advisory lock กัน idempotency race
- lock บัญชีตาม UUID order, กันยอดติดลบ, กันต่าง currency และบังคับยอด postings รวมเป็นศูนย์
- เพิ่ม deferred database constraint guard ป้องกัน direct SQL ที่สร้าง POSTED transaction ไม่สมดุล
- ไม่ติดตั้ง .NET, ไม่เปิด HTTP/Cloud และไม่มีค่าใช้จ่าย

### Regression Tests

- same key + same payload replay transaction เดิมและไม่บวกยอดซ้ำ
- same key + changed payload ถูก reject
- unbalanced direct write ถูก database reject
- concurrent debit 700 + 700 จากยอด 1,000 สำเร็จได้หนึ่งคำขอเท่านั้น; ยอดไม่ติดลบและยอดรวมคงเดิม
- พบและแก้ pgcrypto schema resolution ภายใต้ `SECURITY DEFINER`; test เดิมคุม regression ไว้
